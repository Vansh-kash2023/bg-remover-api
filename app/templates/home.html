Welcome to Background Removal Flask API

  • The API accepts an image link as input and the coordinates for the product area within the image.
  • To use this API,send a POST Request on the- 
    /process-image
    Endpoint with the following request structure :
    • {
                    "image_url":"public_image_url",
                    "bounding_box": {
                        "x_min": ,
                        "y_min": ,
                        "x_max": ,
                        "y_max": 
                    }
                }
  • It wll return the processed image through a unique URL
  • To run the project locally,clone the repository and run the flask app locally to use the API endpoints.
